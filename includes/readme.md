# Includes Folder

This folder contains reusable PHP files shared across the project.

## Files

- `db.php` - MySQL database connection.
- `session.php` - Session timeout and login/session setup.
- `navbar.php` - Shared navigation bar for voter and admin pages.
- `header.php` - Optional reusable page header template.
- `footer.php` - Optional reusable page footer template.

## Notes

These files help avoid repeating common setup code in every page.
