# CSS Folder

This folder contains stylesheets used by the frontend and admin panel.

## Files

- `login.css` - Styling for the login and password recovery pages.
- `register.css` - Styling for the registration page.
- `dashboard.css` - Main shared layout, cards, navbar, forms, result bars, and theme styles.
- `admin.css` - Admin-specific improvements for dashboard cards, forms, lists, and responsive admin screens.

## Notes

The project uses plain CSS only. No CSS framework is required.
