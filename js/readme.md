# JavaScript Folder

This folder contains plain JavaScript used by the project.

## Files

- `main.js` - Theme toggle logic using browser local storage.
- `login.js` - Login page password toggle and message display.
- `register.js` - Registration form validation and password toggle.
- `vote.js` - Vote form confirmation before submitting.
- `charts.js` - Simple result bar animation.
- `validation.js` - Common helper validation functions.
- `dashboard.js` - Reserved for dashboard-specific interactions.

## Notes

The project uses plain JavaScript only. No frontend framework is required.
